# Plan 06-03: Interactive Mermaid Diagram Export

## Objective

Add Mermaid diagram export capability, allowing users to export traces as interactive Mermaid diagrams that can be embedded in documentation or viewed in Mermaid live editor.

## Context

Completing the runtime visualization phase with export functionality for documentation and sharing.

@CONTEXT.md

## Tasks

### Task 1: Create Mermaid generator utilities

**type:** auto

Create `ui/src/utils/mermaidGenerator.ts` with:
- `generateMermaidFlowchart(diagram)` - Generate flowchart syntax
- `generateMermaidSequence(trajectory)` - Generate sequence diagram syntax
- `generateMermaidState(trace)` - Generate state diagram syntax
- `validateMermaidSyntax(code)` - Basic validation
- Escape special characters in labels

```typescript
interface MermaidExport {
  flowchart: string;
  sequence: string;
  state: string;
}

function generateMermaidFlowchart(diagram: DiagramData): string;
function generateMermaidSequence(trajectory: ExecutionTrajectory): string;
function generateMermaidState(trace: SerializedTrace): string;
```

**done when:**
- File created with all generators
- Generated syntax is valid Mermaid
- Handles edge cases (special chars, empty traces)

### Task 2: Create MermaidExport component

**type:** auto

Create `ui/src/components/MermaidExport.tsx` with:
- Diagram type selector (Flowchart/Sequence/State)
- Syntax preview with syntax highlighting

- Download as .mmd file button
- Open in Mermaid Live link

Features:
- Live preview using mermaid.js
- Format options (theme, direction)
- Export history

**done when:**
- Component renders preview
- Copy/download works
- Live editor link functional
