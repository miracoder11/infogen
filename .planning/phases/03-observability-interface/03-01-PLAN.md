# Phase 03 Plan 01: Web UI Foundation

## Objective
Set up web UI infrastructure with Vite + React + TypeScript for trace visualization interface.

## Context
- Phase 2 complete: Trace infrastructure with storage API available at `src/storage/trace-storage.ts`
- Need to expose trace data via HTTP API
- Need web UI foundation for visualization components

## Tasks

### Task 1: Express API Server
**type: "auto"**

Create Express server to expose trace data:

```typescript
// src/api/server.ts
import express from 'express';
import { getTraceStorage } from '../storage/trace-storage.js';

const app = express();
app.use(express.json());

// GET /api/traces - list all traces
app.get('/api/traces', (req, res) => {
  const storage = getTraceStorage();
  const traces = storage.getAll().map(t => ({
    traceId: t.traceId,
    statementId: t.statementId,
    deliverableType: t.deliverableType,
    spanCount: t.spans.length,
    startTime: t.startTime,
    duration: t.duration
  }));
  res.json(traces);
});

// GET /api/traces/:id - get trace details
app.get('/api/traces/:id', (req, res) => {
  const storage = getTraceStorage();
  const trace = storage.get(req.params.id);
  if (!trace) return res.status(404).json({ error: 'Not found' });
  res.json(trace);
});
```

**done:**
- Express server running on port 3001
- `/api/traces` endpoint returns trace list
- `/api/traces/:id` returns trace details
- Server starts with `npm run server`

### Task 3: Basic Trace List Component
**type: "auto" tdd: "true"**

Create TraceList component to fetch and display traces:

**behavior:**
- Fetches traces from `/api/traces` on mount
- Displays trace cards with: traceId (truncated), statementId, spanCount, duration
- Shows loading state while fetching
- Handles empty state

**implementation:**
```tsx
// ui/src/components/TraceList.tsx
interface Trace {
  traceId: string;
  statementId?: string;
  spanCount: number;
  duration: number;
}

export function TraceList() {
  const [traces, setTraces] = useState<Trace[]>([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch('/api/traces')
      .then(r => r.json())
      .then(data => { setTraces(data); setLoading(false); });
  }, []);

  if (loading) return <div>Loading traces...</div>;
  if (traces.length === 0) return <div>No traces yet</div>;

  return (
    <div className="trace-list">
      {traces.map(t => (
        <TraceCard key={t.traceId} trace={t} />
      ))}
    </div>
  );
}
```

**tests:** `ui/src/components/__tests__/TraceList.test.tsx`
- renders loading state initially
- displays traces after fetch
- shows empty state when no traces
