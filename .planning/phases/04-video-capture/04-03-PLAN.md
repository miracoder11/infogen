# Plan 04-03: Video Annotation and Post-Processing

## Objective

Add timestamp annotations, transitions, and post-processing capabilities to recorded videos.

## Context

@04-02-PLAN.md - Automation layer complete. Now add video enhancement features.

## Success Criteria

- Videos can include timestamp annotations
- Basic transitions available (fade in/out)
- Video metadata includes recording info

## Tasks

### Task 1: Implement Annotation System
type: auto

**Action:**
1. Create `src/video/annotations.ts`
2. Define annotation types (timestamp, text, highlight)
3. Implement annotation timeline builder
4. Integrate with recorder for real-time annotation

**Implementation:**
```typescript
export interface VideoAnnotation {
  timestamp: number;
  type: 'timestamp' | 'text' | 'highlight';
  content: string;
  duration?: number;
}

export class AnnotationTimeline {
  addAnnotation(annotation: VideoAnnotation): void
  getAnnotations(): VideoAnnotation[]
  toJSON(): string
}
```

**Done when:**
- Annotations can be added and serialized

---

### Task 2: Implement Video Post-Processor
type: auto

**Action:**
1. Create `src/video/post-processor.ts`
2. Use FFmpeg to add annotations as overlays
3. Add fade transitions at start/end
4. Optimize video encoding settings

**Implementation:**
```typescript
export class VideoPostProcessor {
  async process(input: string, output: string, options: ProcessOptions): Promise<void>
  async addAnnotations(video: string, annotations: VideoAnnotation[]): Promise<string>
  async addTransitions(video: string, fadeIn: number, fadeOut: number): Promise<string>
}
```

**Done when:**
- FFmpeg commands execute correctly
- Processed videos are playable

---

### Task 3: Implement Video Metadata
type: auto

**Action:**
1. Create `src/video/metadata.ts`
2. Store recording metadata (duration, resolution, timestamp, annotations)
3. Generate metadata JSON sidecar file
4. Include trace ID linkage for observability

**Implementation:**
```typescript
export interface VideoMetadata {
  id: string;
  filename: string;
  duration: number;
  resolution: { width: number; height: number };
  recordedAt: string;
  traceId?: string;
  annotations: VideoAnnotation[];
}
```

**Done when:**
- Metadata is saved alongside videos

---

### Task 4: Create Video Processing Pipeline
type: auto

**Action:**
1. Create `src/video/pipeline.ts`
2. Orchestrate: record -> annotate -> post-process -> metadata
3. Handle errors at each stage
4. Provide progress callbacks

export class VideoPipeline {

**Done when:**
