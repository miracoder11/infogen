# MCP Server for Trace/Video APIs

## Objective
Create an MCP (Model Context Protocol) server that exposes trace and video APIs for Claude Code integration.

## Context
- Phase 3 provides trace viewing API
- Phase 4 provides video recording API
- MCP server allows Claude Code to call these APIs

## Tasks

### Task 1: Create MCP server package
Create `src/mcp-server/` with:
- `index.ts` - server entry point
- `tools/` - MCP tool definitions
- `resources/` - MCP resource definitions

### Task 2: Implement trace tools
Tools for trace operations:
- `list_traces` - list available traces
- `get_trace` - get trace by ID
- `search_traces` - search traces by filter

### Task 3: Implement video tools
Tools for video operations:
- `list_videos` - list available videos
- `get_video` - get video metadata
- `start_recording` - start video recording
- `stop_recording` - stop video recording

### Task 4: Implement artifact tools
Tools for artifact operations:
- `list_artifacts` - list collected artifacts
- `get_artifact` - get artifact by ID
- `link_artifact` - link artifact to statement

### Task 5: Create MCP configuration
Add MCP server configuration to `.claude.json`

## Verification
- MCP server starts successfully
- Tools are discoverable
- API calls return expected data

## Output
- `src/mcp-server/` directory
- Updated `.claude.json` with MCP config

## Done Criteria
- [ ] MCP server package created
- [ ] All tools implemented
- [ ] Configuration added
- [ ] Server can be started
