# Phase 5: Claude Code Integration

## Goal
Connect perceptible delivery to Claude Code via skills and MCP

## Context
This is an infrastructure phase. Assumes:
- Phase 3: Observability Interface is complete (trace viewing UI)
- Phase 4: Video Capture is complete (Playwright recording)

## Key Deliverables
1. Claude Code skills for triggering perceptible delivery
2. MCP server exposing trace/video APIs

## Success Criteria
1. User can trigger demo recording via Claude Code skill
2. User can declare demo statements through Claude Code commands
3. Verification artifacts automatically collected during agent execution
4. User can view collected artifacts after session completion
