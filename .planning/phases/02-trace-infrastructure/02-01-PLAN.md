Create .env.example with OpenTelemetry environment variables:

```bash
# OpenTelemetry Configuration

# OTLP Trace Exporter Endpoint
# Default: http://localhost:4318/v1/traces (OTel Collector default)
OTEL_EXPORTER_OTLP_TRACES_ENDPOINT=http://localhost:4318/v1/traces

# OTLP Authentication Token (optional)
# OTEL_AUTH_TOKEN=your-token-here

# Service Name (appears in trace backend)
OTEL_SERVICE_NAME=infogen-agent

# Service Version
OTEL_SERVICE_VERSION=1.0.0

# Sampling Rate (0.0 to 1.0)
# 1.0 = record all traces, 0.1 = record 10% of traces
OTEL_SAMPLING_RATE=1.0

# Export timeout in milliseconds
OTEL_EXPORTER_OTLP_TIMEOUT=10000

# Enable console logging for exporter errors
OTEL_LOG_LEVEL=info
```
